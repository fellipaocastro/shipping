Shipping
========

Shipping calculator.

Requirements
------------

- Python 2.7.8
- pip 1.5.6

Setup
-----

.. code-block:: bash

    $ shipping/manage.sh setup

Usage
-----

.. code-block:: bash

    $ shipping/manage.sh run <origin> <destination> <receipt> <weight>

Test
----

.. code-block:: bash

    $ shipping/manage.sh test

Source code check
-----------------

.. code-block:: bash

    $ shipping/manage.sh check
